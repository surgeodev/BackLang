BackLang

Langage de script rapide avec support SQLite, serveur HTTP intégré, et débogueur VS Code.

Installation en une commande :
curl -fsSL https://raw.githubusercontent.com/surgeodev/BackLang/main/install.sh | bash

Installation

# Automatique (Linux)
curl -fsSL https://raw.githubusercontent.com/surgeodev/BackLang/main/install.sh | bash

# Manuel
sudo apt install libsqlite3-dev pkg-config build-essential
cargo build --release
sudo cp target/release/bl /usr/local/bin/bl

Utilisation

bl --check file.bl        # Vérifier la syntaxe
bl file.bl                # Exécuter
bl --debug file.bl        # Déboguer (VS Code)

Bibliothèque standard

std.db — SQLite

import "std.db"
let db = db.open("/tmp/data.db")
db.execute("CREATE TABLE IF NOT EXISTS users (id INT, name TEXT)")
let rows = db.query("SELECT * FROM users")

std.os — Système

import "std.os"
let home = std.os.getenv("HOME")
std.os.exit(0)
let args = std.os.args()

std.fs — Fichiers

import "std.fs"
let content = std.fs.readFile("data.txt")
std.fs.writeFile("out.txt", "hello")

std.math — Maths

import "std.math"
print(std.math.sqrt(25))     // 5
print(std.math.abs(-10))     // 10
print(std.math.floor(3.7))   // 3
print(std.math.ceil(3.2))    // 4

std.string — Chaînes

import "std.string"
let parts = std.string.split("a,b,c", ",")
print(std.string.trim("  hi  "))   // "hi"
print(std.string.toUpper("hi"))    // "HI"
print(std.string.toLower("HI"))    // "hi"

std.random — Aléatoire

import "std.random"
let r = std.random.rand()
let d = std.random.randint(1, 6)   // dé 1-6

Serveur HTTP

server app { port: 8080; cors: true }

endpoint GET "/api/hello" {
    return {status: 200, body: {message: "Hello"}}
}

endpoint POST "/api/data" {
    return {status: 200, body: {received: req.body}}
}

return {status: N, body: ...} définit le code HTTP et le body JSON.

SQLite intégré

import "std.db"

let dbPath = "/tmp/app.db"
db.open(dbPath)
db.execute(dbPath, "CREATE TABLE IF NOT EXISTS items (id INTEGER, name TEXT)")

endpoint GET "/api/items" {
    return {status: 200, body: db.query(dbPath, "SELECT * FROM items")}
}

endpoint POST "/api/items" {
    db.execute(dbPath, "INSERT INTO items VALUES (1, 'test')")
    return {status: 201, body: {ok: true}}
}

Extension VS Code

L'extension BackLang offre :

  • Coloration syntaxique
  • Complétion automatique (mots-clés, builtins, db.*)
  • Documenation au survol
  • Inlay hints (types inférés)
  • CodeLens (▶ Run / Start Server)
  • Folding, symbols, rename, references
  • Diagnostics en temps réel
  • 12 snippets
  • Débogueur interactif

Débogueur

Lancez le débogueur depuis VS Code (F5) ou en CLI :

bl --debug file.bl

Supporte : breakpoints, step over, step in, step out, continue.
